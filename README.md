# Site web — BerettiPrenant

Site web multi-pages réalisé dans le cadre du cours de **Programmation Web**
(Licence MIASHS). HTML / CSS statique, sans framework.

## Aperçu

Petit site éditorial : page d'accueil, articles et page de contact, avec une
feuille de style maison.

## Structure

| Fichier | Rôle |
|---------|------|
| `index.html` | page d'accueil |
| `article1.html`, `article2.html` | articles |
| `contact.html` | page de contact |
| `style.css` | feuille de style |
| `images/` | ressources visuelles |

## Voir le site

Ouvrir `index.html` dans un navigateur (double-clic), ou servir le dossier :

```bash
python -m http.server 8000   # puis ouvrir http://localhost:8000
```

## Compétences mises en œuvre

HTML sémantique, CSS (mise en page, responsive), structure multi-pages, navigation.
